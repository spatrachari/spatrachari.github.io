A Few School Projects...

Song Search Query | Data Structures Class Project

Worked in a partnership and used advanced C++ knowledge to develop a program that performs query specific searches within a large database of songs. The program completes partial and whole word searches within song names, artist names, album names, and song ID numbers to narrow search results to information relevant to the user. We implemented hash tables of pointers to enhance the efficiency of the find operation; we also created deep copies of object to later return all song information on program completion. To debug, we used the gdb tool.

E Language Parser | Java Programming Class Project

Worked with three additional students to create a java program that converts the arbitrary E language to C. We encoded the original E language grammar to scan and tokenize the elements in our test files. By parsing though the input language to validate variable names through searches within the local and global scopes of our symbol table, we translated the tokens and variable names into the appropriate C equivalent code

Machine Learning Digit Detection | Applied Linear Algebra Programming Project

Pattern recognition seeks to perform computations to yield higher recognition rates for the visual patterns of interest. In our algorithm we used a data set of handwritten digits from the U.S. Postal Service Database. Both the arrays of training data, trainpatterns and testpatterns, are image reconstructions represented by 16x16 pixel intensities. I constructed an alogrithm to iteratively learn and classify these handwritten digits.

Projects for Fun!

Space Updates Amazon Echo App | HackDavis

With a team of two other members at a Hackathon, we developed a lambda application on AWS using the NASA and Twilio API to allow users to prompt an Amazon Echo for daily alerts and random facts about space. We encoded different speech responses in the application, so the user could deliver the Echo more insight into what information they'd like to recieve and what number they'd like the information sent to. To inform the user about the status of the request, and it's validity, we programmed different responses the Echo device would trigger.
